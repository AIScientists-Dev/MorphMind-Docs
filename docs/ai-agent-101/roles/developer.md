# Data Science Agents

Agents for data analysis workflows, supporting natural language interaction with datasets.

**Intended users:** Data scientists, analysts, researchers working with data

## Available Agents

### Talking-to-Data Agent

Enables natural language queries on datasets.

**Capabilities:**
- Accept datasets in common formats (CSV, Excel, databases)
- Respond to natural language questions about data
- Generate visualizations and statistical analyses
- Trace outputs back to source data

**Typical uses:**
- Exploratory data analysis
- Quick insights from unfamiliar datasets
- Report generation
- Hypothesis validation

**Example queries:**
- "What are the top 5 factors correlated with customer churn?"
- "Show me the monthly trend of sales by region"
- "Are there any outliers in this dataset?"

---

### Financial / Stock Analysis Agent

Analyzes financial data and market trends.

**Capabilities:**
- Analyze stock performance and trends
- Compare companies using financial metrics
- Generate research reports
- Contextualize market movements

**Typical uses:**
- Investment due diligence
- Market research
- Portfolio analysis
- Financial reporting

---

## Key Features

### Natural Language Analysis

Users can interact with data without writing code:

1. Upload dataset
2. Describe analysis goals
3. Review proposed approaches
4. Select and refine methods
5. Export results

### Visualization

Generate charts and figures through conversation:

- Statistical plots (box plots, violin plots, scatter matrices)
- Time series visualizations
- Geographic maps
- Custom dashboards

### Methodology Transparency

Each analysis is:
- **Traceable** — View exact calculations and transformations
- **Reproducible** — Export methodology for review
- **Editable** — Modify steps in the pipeline

## Preference Learning

Agents can learn user preferences over time:

- Visualization formats
- Preferred statistical methods
- Reporting templates
- Domain terminology

This reduces the need to repeat instructions across sessions.
