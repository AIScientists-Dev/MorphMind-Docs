# Agents by Domain

Different disciplines have distinct workflows and terminology. This section organizes available agents by domain to help you find relevant starting points.

## Browse by Domain

### [Biomedical Research](./researcher)

Agents for biology and medicine research workflows, including literature review, single-cell analysis, and clinical interpretation.

**Available agents:**
- Literature Search and Reasoning Copilot
- scRNA-seq Copilot
- Spatial Transcriptomics Copilot
- Multi-omics Copilot
- Clinical Translation Agent

## Selecting an Agent

1. **Identify your domain** — Browse the category matching your field
2. **Review available agents** — Find ones addressing your specific tasks
3. **Customize if needed** — Agents can be adapted to your preferences
4. **Create new agents** — Use AgentLab if existing agents don't fit your needs

## Building Custom Agents

If no existing agent matches your workflow, AgentLab allows you to create custom agents:

- Describe your workflow in natural language
- Upload relevant materials and examples
- Customize using your domain knowledge
- Share with team members
